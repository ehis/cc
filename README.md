# CC

[cc.shud.in](https://cc.shud.in): Shu's [Creative Coding](https://works.shud.in/creative-coding) template.

The code is open sourced under MIT, feel free to use it to create your own art.

## Usage

With node and yarn:

```
yarn
yarn dev
```

Or npm:

```
npm install
npm run dev
```

Then open localhost:3000.
